Call the api

You can do the same with any image by using the following code. Be sure to replace the path with the path of your image.

import requests
import keras
import json
url = 'http://127.0.0.1:5000/predict'
my_img = {'image': open('PUT_YOUR_PATH_HERE', 'rb')}
r = requests.post(url, files=my_img)

data = json.loads(r.json()['data'])

keras.preprocessing.image.array_to_img(
data, data_format=None, scale=True, dtype=None,
)
